djangoFirstTouch
================

Django first touch

Для старта проекта нужно запустить:
```
$ ./build_end.sh
```

и после этого всего сделать:
```
$ source ./env/bin/activate
```

после чего строка приглашения станет:
```
<DFT> user:djangoFirstTouch$
```

Для запуска проекта достаточно запустить:
```
./run_dev.sh
```
